# How to use!
## Chromebook
1. Press ctrl + shift + o to open up bookmark manager
2. Copy and paste the code from code.txt
3. Whenever you want to open Noobs GUI simply open the bookmark!
## Ipad
### Safari
1. Go to any website, even the home page
2. Click the button on the left side of where you put in the url
3. Click on the 3 dots
4. Click "Add Bookmark"
5. Rename it and copy the code from code.txt to the url
### Chrome
1. Click the 3 dots
2. Click "Add to Bookmarks"
3. When you click it, click the edit button at the bottom
4. Rename it and change the url to the code from code.txt
